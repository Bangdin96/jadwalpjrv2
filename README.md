# Jadwal PJR UPA TIK

Sistem Penjadwalan Penanggung Jawab Ruangan untuk UPA TIK.

## Fitur
- Jadwal otomatis berdasarkan hari kerja
- Foto PJR untuk setiap lab
- Admin panel untuk mengelola hari libur
- Responsive design

## Tech Stack
- Frontend: HTML, CSS (Tailwind), JavaScript
- Backend: Netlify Functions
- Database: MongoDB Atlas
- Hosting: Netlify
